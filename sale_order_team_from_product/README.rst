This module allow to set sales teams in quotations based on the sales
team defined in the products using the product_sale_team module.

**Table of contents**

.. contents::
   :local:

Use Cases / Context
===================

Sometimes you need to set a product specific sales team in quotation. Eg
the product you are selling requires some specific knowledge and thus a
dedicated team must be in charge of the sales process.

Configuration
=============

- Go to "Sales > Configuration > Settings" and scroll down to the
  "Quotations & Orders" section.
- Activate the "Set Sales Team from Products" option.
- After activation configure the following feature-specific settings:

  - Skip if team is already set. Will not set team from products if it's
    already set in quotation.

Usage
=====

- Create a new quotation.
- Add products with sales team defined. If all products have the same
  sales team this sales team will be set as a sales team for the
  quotation.

Known issues / Roadmap
======================

Allow to select different algorithms for team selection. Eg "Take from
the first line".
